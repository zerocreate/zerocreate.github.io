2.2.2）起步依赖 spring-boot-starter-xx

起步依赖都是以spring-boot-starter-xx作为命名的。起步依赖就像品牌机，买来可以直接用。而自己组装的话需要不仅需要列出一堆零件清单，还可能会遇到零件组合不匹配的问题。比如我们之前要实现web功能，引入了spring-boot-starter-web这个起步依赖即可，它里面已经封装好了实现web功能的需要的全部依赖。

2.2.3）插件spring-boot-maven-plugin

spring-boot-maven-plugin插件主要的两个功能：

  • 1 将项目打包成超级JAR包
    spring-boot-maven-plugin插件是将springboot的应用程序打包成超级JAR包。
    普通的JAR包本质上是将所有.class文件、资源文件(resources目录下)整合成一个包。但是超级JAR包项目依赖的其他JAR也包含进来（包括内嵌容器等），因此具备了独立运行的条件。

    可以运行mvn package命令将项目打包成一个可以直接运行的超级JAR包，使用java -jar jar包名称命令就可以直接运行。

  • 2 定位程序启动的入口类，通过搜索public static void main()方法来标记为可运行类。
